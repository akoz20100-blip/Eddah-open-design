EDDAH / عدة — Visual Strategy

Brand position

EDDAH / عدة is a premium, local-first home maintenance service focused on حي لبن in Riyadh.

The website should feel like a high-end local Saudi service brand, not a generic repair-service template.

Personality

* قريب
* سريع
* منظم
* موثوق
* راقٍ
* واضح
* محلي
* عملي
* حديث

Visual direction

Direction name:
Light Saudi Premium Local

The design should be:

* bright
* warm
* premium
* airy
* refined
* clean
* locally rooted in حي لبن

Avoid:

* dark cinematic dominance
* generic SaaS look
* stock image feel
* overused gradients
* basic rectangular service cards
* cluttered layouts

Color source

The color palette must come primarily from:

/brand/laban-map.png

Use:

* ivory
* warm off-white
* soft beige
* sand
* light stone
* amber/orange from the map pin
* charcoal text
* subtle copper/brass if needed

The orange should be used as an accent, not as a full-page dominant color.

Image system

Use the real EDDAH images from:

eddah-landing/public/brand

Suggested mapping:

* hero technician/tool bag image → hero or trust introduction
* plumbing repair image → السباكة
* electrical maintenance image → الكهرباء
* AC maintenance image → التكييف
* tools-showcase image → craftsmanship / visual trust
* precision-workshop image → precision / quality section
* laban-map image → حي لبن local section
* logo-eddah → navbar/footer/brand moments

Images must be cropped intentionally and integrated into the layout, not dropped into boxes.

Services section direction

The services section must feel premium.

Avoid:

* three equal generic boxes
* basic image-on-top-card-bottom-text layout
* flat Bootstrap-style cards

Use:

* staggered layout
* layered cards
* overlapping panels
* image masks
* floating service badges
* depth shadows
* map-line details
* warm glass panels
* editorial spacing

Hero direction

The hero must communicate quickly:

* حي لبن
* trusted technician
* clear service
* WhatsApp CTA
* speed and warranty

It should not be only text beside an image.
It should feel composed and cinematic but light.

Map section direction

The map section is a main brand section.

It should communicate:

* متخصصين في حي لبن
* فني قريب منك
* وصول أسرع
* تنظيم أفضل للمواعيد

Use the real map image prominently.

Motion direction

Motion should be inspired by premium Framer websites:

* smooth
* restrained
* elegant
* purposeful
* not excessive

Use:

* hero entrance
* card stagger
* hover depth
* image reveal
* sticky nav transition
* reduced-motion support

Mobile direction

Mobile must be treated as primary.

On mobile:

* hero should convert quickly
* CTAs must be easy to reach
* service cards must feel premium
* images must not become awkward crops
* sections must have strong rhythm
* text must be readable
