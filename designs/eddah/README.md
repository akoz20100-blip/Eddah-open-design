## ما هو المشروع

صفحة هبوط عربية RTL فاخرة لعلامة **عدّة / EDDAH** — خدمة صيانة منزلية محلية في **حي لبن، الرياض**.
التحويل عبر **واتساب**. ثلاث خدمات فقط: **السباكة · الكهرباء · التكييف**.

- **الستاك:** Next.js 14 (App Router) · React 18 · TypeScript · Tailwind CSS 3 · framer-motion 11
- **بدون أي dependency للرسوميات** — خلفية «سديم» مبنية بـ Canvas محلي.
- **خط الهوية:** خط ثمانية (Thmanyah) — Sans للنصوص، Serif Display للعناوين.
- **وضعان:** نهاري (Ivory & Amber) ومسائي (أسود + بنفسجي خفيف + كلمات برتقالية).

## ملخص سريع جدًا (TL;DR)

- **الألوان نهاري:** خلفية `#FCF8F1` · نص `#1A1714` · أكسنت برتقالي `#F2820C` · ذهبي `#F6A700`.
- **الألوان مسائي:** خلفية `#030305` (أسود) · نص `#F5F2FF` · أسطح `#0C0B11` · أكسنت برتقالي نفسه · لمسة بنفسجي `#7C4DFF` في الخلفية فقط.
- **الخط:** Thmanyah Sans (نصوص) + Thmanyah Serif Display (عناوين H1/H2).
- **الحاوية:** `max-width: 1180px` · padding `24px`/`40px`.
- **الزوايا:** كبيرة (`1.5rem`–`2.75rem`).
- **الحركة:** easing واحد `cubic-bezier(0.16, 1, 0.3, 1)`.
- **الخلفية:** نيبولا «سديم» على Canvas — كهرماني نهارًا، بنفسجي خافت ليلًا.
- **التحويل:** واتساب `966509005845`.
