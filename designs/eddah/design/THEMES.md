# الوضع النهاري/المسائي — THEMES

تبديل وقت التشغيل بدون إعادة بناء، عبر متغيّرات CSS + سمة `data-theme` على `<html>`.

## كيف يعمل

1. **الألوان كمتغيّرات:** في `tailwind.config.ts` كل لون = `rgb(var(--token) / <alpha-value>)`.
   فالمكوّنات تستخدم `bg-clay-50`, `text-ink`, `text-orange-500` كالعادة، لكن قيمها تأتي من CSS.
2. **مجموعتا قيم في `globals.css`:**
   - `:root { … }` = الوضع النهاري (افتراضي).
   - `[data-theme="night"] { … }` = يستبدل القيم للوضع المسائي.
3. **الزر `ThemeToggle`:** يضيف/يزيل `data-theme="night"` على `document.documentElement`،
   يحفظ في `localStorage` (`eddah-theme`)، ويُطلق `window` event باسم `themechange`.
4. **منع وميض الإقلاع:** سكربت سطري في `layout.tsx` (قبل المحتوى) يقرأ `localStorage`
   ويضبط `data-theme` قبل أول رسم.
5. **خلفية «سديم»** تستمع لـ `themechange` وتبدّل لوحتها (كهرماني ↔ بنفسجي) فورًا.

## القيم (مختصر)

| التوكِن | نهاري | مسائي |
|---|---|---|
| `--clay-50` (الخلفية) | `252 248 241` (#FCF8F1) | `3 3 5` (#030305) |
| `--white` (الأسطح) | `255 255 255` | `12 11 17` (#0C0B11) |
| `--ink-900` (النص) | `26 23 20` | `245 242 255` |
| `--orange-500` (الأكسنت) | `242 130 12` | `242 130 12` |
| `--orange-600` (نص أكسنت) | `221 106 6` | `251 166 58` |
| `--orange-50` (خلفية أيقونة) | `255 245 225` | `20 14 34` (بنفسجي داكن) |
| `--shadow` | `26 23 20` | `0 0 0` |

القيم الكاملة في `code-snippets/globals.css`.

## للتطبيق في مشروع جديد
انسخ كتلة المتغيّرات من `globals.css`، واستخدم صيغة `rgb(var(--x) / <alpha-value>)`
في `tailwind.config.ts`، وأضف `ThemeToggle.tsx` + السكربت السطري. خلاص.
