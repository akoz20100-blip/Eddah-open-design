## 2) الخطوط — خط ثمانية (Thmanyah)

- **Thmanyah Sans** → النصوص والواجهة والأزرار · متغيّر `--font-arabic` · أوزان 300/400/500/700/900.
- **Thmanyah Serif Display** → العناوين الكبيرة (H1, H2) · متغيّر `--font-display` · إحساس تحريري فاخر.
- محمّلة عبر `next/font/local` من `public/fonts/thmanyah/*.woff2` (انظر `code-snippets/layout.tsx`).
- القاعدة: `h1, h2 { font-family: var(--font-display) … }` — البقية (h3، النصوص) Sans.
- **العناوين:** `tracking: -0.01em` (تتبّع لطيف يناسب الحرف العربي المتّصل — لا تستخدم تتبّعًا سالبًا قويًا).
- **المقياس:** H1 `clamp(2.6rem, 7vw, 4.7rem)` · H2 `clamp(1.875rem, 3.8vw, 44px)`.
- أرقام عربية (٠١ · ٢ · ٣) عمدًا للأصالة. RTL كامل (`dir="rtl"`).
- الملفات: `fonts/thmanyah/*.woff2` (+ `thmanyah-otf/` الأصلية) + ملف الترخيص.

## 3) المسافات والإيقاع

- حاوية واحدة: `max-width: 1180px` · padding أفقي `24px` (موبايل) / `40px` (md). انظر `Container`.
- أقسام: `py-20 md:py-28` (وأحيانًا `py-32` للبطل) — فراغ سخيّ = فخامة.
- زوايا موحّدة كبيرة: `rounded-[1.5rem | 2rem | 2.25rem | 2.75rem]` (`rounded-4xl=2rem`, `rounded-5xl=2.75rem`).
- شبكة الخدمات **غير متناظرة** (bento split-panels) لا بطاقات متساوية.
- **تقليل «كروت AI»:** الأقسام التحريرية تستخدم قوائم بحدود شعرية (hairline) بدل صناديق مكرّرة.

## 5) المكوّنات الرئيسية

- **Hero:** عنوان serif بكشف كلمة-بكلمة (wordReveal) + صورة فني بـ parallax + بطاقات عائمة (مباشر/تأكيد الموعد).
- **Services:** bento تحريري غير متناظر — السباكة مميّزة + الكهرباء/التكييف، شِب زجاجي + رقم شبحي + أيقونة عائمة.
- **MapSection:** صورة `laban-map.png` كقسم رئيسي + دبوس متوهّج + لوحة معلومات زجاجية.
- **ValueProps:** تخطيط تحريري بعمودين (عنوان ثابت sticky + قائمة ٠١–٠٦ بلا صناديق).
- **Trust / LocalFocus:** عناصر بلا صناديق، فواصل/خطوط أكسنت بدل البطاقات.
- **Testimonials:** اقتباسات بحدّ جانبي (border-s) لا بطاقات.
- **CTASection / StickyWhatsApp / Nav (+ زر الوضع):** تحويل واتساب واضح.
- **الزجاج:** `bg-white/80 backdrop-blur-xl border border-white/70`.
- **الشِبس:** `rounded-full border bg-white/80`.
- **الأزرار:** `MagneticButton` (primary/ghost/whatsapp) — يُسحب نحو المؤشر.

ترتيب الأقسام: `Nav → Hero → StatsBand → MapSection → HowItWorks → ValueProps → Services → Showcase → LocalFocus → Testimonials → Trust → CTA → Footer`.

## 6) الحركة (motion.ts)

| المتغيّر | الاستخدام |
|---|---|
| `EASE_OUT = [0.16, 1, 0.3, 1]` | كل الحركة (expoOut) |
| `fadeUp` / `fadeUpSmall` | ظهور مع/بدون blur |
| `wordReveal` | كشف العناوين كلمة بكلمة |
| `scaleIn` · `staggerContainer` · `inViewProps` | بقية الحركات |

قواعد: بدون مكتبات إضافية (لا Lenis/GSAP) · كل شيء يحترم `prefers-reduced-motion` · parallax خفيف للصور عبر `useScroll + useTransform`.
