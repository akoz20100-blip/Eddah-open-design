# عدة / EDDAH — Design Docs (مرجع التصميم)

مجلد **مستقل ومخصّص** لتصميم موقع عدة فقط. هذا هو المكان اللي ترجع له مستقبلاً
لو بغيت تعدّل التصميم أو تطلب إعادة تصميم — بدون ما تلخبط مع ملفات المشروع القديمة
(`AGENTS.md`, `CODEX.md`, `DESIGN_STRATEGY.md`, `HANDOFF.md`). تلك الملفات تبقى مكانها؛
هذي نسخة مرتّبة وموحّدة للرجوع السريع.

## الملفات

| الملف | المحتوى | استخدمه لـ |

## مرجع سريع (الأساسيات)
- **العلامة:** عدة / EDDAH — صيانة منزلية محلية في **حي لبن، الرياض**.
- **الخدمات (٣ فقط):** السباكة · الكهرباء · التكييف. لا تضيف غيرها أبداً.
- **الإحساس:** فاتح، دافئ، فاخر، محلي سعودي، منظّم، موثوق — **مو داكن، مو generic**.
- **مرساة اللون والمحلية:** `public/brand/laban-map.png`.
- **التحويل:** واتساب أولاً.
- **العنوان:** «فنيك في حيك، تواصل وأبشر بالسعد».

> ملاحظة: معلومات هذي الملفات مستخرجة من الكود الفعلي للمشروع (المصدر الأدق).
> رابط Claude Design الخاص لا يمكن الوصول له برمجياً (محمي بالحساب)؛ لدمج محتواه الصقه يدوياً.
